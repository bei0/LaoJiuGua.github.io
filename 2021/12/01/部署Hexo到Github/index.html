1.准备条件

  • GitHub账号
  • 域名(可选)

2.新建Github项目

首先在 GitHub 新建一个仓库（Repository），名称为 {username}.github.io，注意这个名比较特殊，必须要是 github.io 为后缀结尾的。比如我的用户名是LaoJiuGua，那么就新建一个LaoJiuGua.github.io的仓库。

3.配置SSH-Key

为了提交代码是还需要登录验证，如果不嫌麻烦可以不用配置，度娘搜索Github配置SSH-Key教程一大把，这里就不详细说了

下载地址：https://nodejs.org/zh-cn/download/，可以安装 Stable 版本。
